В DIPLOM_LAB_STAND_SCHEMES находятся принципиальные схемы стенда и разводки плат в вариантах с модулями (с использованием заливки земли полигоном ..._gnd или без заливки) и без них (тоже с заливкой полигоном и без неё).

DIPLOM.kicad_sch - Принципиальная схема управляющей платы.

пс_плата_2.kicad_sch - Принципиальная платы расширенной периферии.

DIPLOM.kicad_pcb - Разводка управляющей платы.

DIPLOM_BOARD2.kicad_pcb - Разводка платы расширенной периферии.

В DIPLOM_LAB_STAND находятся файлы проекта для Microchip Studio. В DEBUG находится .hex файл для прошивки микроконтроллера. DIPLOM_LAB_STAND.atsln - сам проект.
